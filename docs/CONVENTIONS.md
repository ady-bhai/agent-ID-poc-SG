# Project Conventions

**Default norms for this project.**

---

## Party Mode & Multi-Agent Discussion Outcomes

**Norm:** When a party mode or multi-agent discussion produces actionable recommendations, decisions, or roadmaps, capture them in a doc under `docs/`.

**Format:** `docs/0X-{TOPIC}-{DESCRIPTIVE-NAME}.md`  
**Example:** `docs/03-PARTY-MODE-BUILD-ROADMAP.md`

**Contents should include:**
- Date and context
- Agent perspectives (who said what)
- Consolidated checklist or recommendations
- Cross-references to source docs

**Why:** Ensures insights from collaborative sessions are referenceable and don't get lost.

---

## Doc Numbering

- `00-` — Indexes, source lists
- `01-` — Synthesis, analysis, evaluation
- `02-` — Stress tests, criticism, unasked questions
- `03-` — Party mode / multi-agent outcomes, roadmaps
- `04+` — Future extensions
